发表于 2020-10-06

对画图的人来说有这样的参考书真是谢天谢地，一迅社推出了「特殊画法系列」的绘图教学书《各种胸部画法》，里面的内容包含挤压的胸部…各种车头灯….还有跳动时的胸部，老实讲光是用想的就很难想向该怎幺画这些东西，现在有个参考资料对绘师来说可能是一大福音？

参考书《各种胸部画法》

《各种胸部画法》具体来说还会告诉读者乳晕的大小、乳头的形状、着衣时的胸部等等，都是一些不怎幺好参考的特殊玩意儿，由于这些练习内容要寻找参考资料相当不容易，因此这本书目前登上日本亚马逊的热销排行。

里面还有跳动时候的胸部（团队似乎有实际验证）

一迅社的「特殊画法系列」还有《各种内裤画法》，就跟胸部一样，里面有各种描绘内裤的画法教学，像是拉内裤、蹲坐在地上的时候、或是各种角度的内裤等等….

工具书《各种内裤画法》

不过第一本《各种内裤画法》得到的买家评价不是说很高，由于内容较为针对初学者，不知道下一本会不会在内容上面做些调整，但念在工具书题材相当有创意性，我想还是不少绘师会想买来翻看看XD”

这本《各种胸部画法》售价为1836日币，预计4月30日发售，有在画图的朋友应该可以考虑看看。（商品页面在此）

发表于 2020-10-06

日本儿童向动画大多安排在星期日早上或平日傍晚播出，作品範围囊括变身美少女以及卡片对战等题材。不过最近却有日本网友主张充斥性暗示的儿童向动画越来越多，家长根本无法安心让小孩收看这些节目。此话一出自然引起其他网友的议论，其实比起古早时代的晨间动画，现在的规制已经严格很多了。而且会觉得动画很猥亵，会不会是因为自己的思想已经不再像儿时那般纯粹了呢？

被点名充斥性暗示的两部作品

今年2月开播的光之美少女最新系列《魔法使 光之美少女！》

《魔法使 光之美少女！》收视族群主要针对低学年女童，主题承袭了过去系列作的爱、友情、梦想等重点。同时藉由故事传达即使有了方便的魔法也不是万能的，最重要的还是人之人的内心交流。虽然会出现变身画面或女角大腿等画面，但基本上都是剧情需要并不会流于情色。

《魔法使 光之美少女！》变身画面

《12岁》改编自漫画家「まいた菜穂」所创作的少女漫画，描写12岁这个年龄层的少女在人际关係上常会遇到的烦恼。从青少年身心理的角度来看，是一部值得探讨的作品。

《剧作家大川久男去世》作品：阿尔卑斯山的少女、海底两万哩

发表于 2020-10-06

许多日本早年动画业界的大人物都在最近离开人世，就在今天早上，日本朝日新闻报导日本动画剧本家大川久男，在本月19日因为呼吸衰竭离开人世，或许认识这位人物的朋友并不是这幺的多，过去他曾经担任过《阿尔卑斯山的少女》《海底两万哩》等名作编剧，可以说是动画业界早期的重要推手。

大川久男担任多部早年知名作品，在动画业界的重要地位不言而喻。

《山老鼠洛奇查克》

动画片段

这一部作品是1973年的儿童向动画，故事描写一群拟人化的动物在森林里面的生活，动画《白箱》的剧中剧《安地斯恰吉》就是在影射这一部动画作品，由于这一部作品在台湾好像没有播出（？），加上年代久远，可能有印象的人不是这幺多（笔者自己也没看过）

《白箱》的剧中剧《安地斯恰吉》

角色构成也非常相像

可以看到相似度非常高，水岛努的致敬意味浓厚。

《阿尔卑斯山的少女》（别名：小天使）

知名片段：克拉拉站起来了！

和上面的洛奇查克相比，《阿尔卑斯山的少女》就比较多人知道，导演是高畑勋，这部作品也是让日本动画扬名海外的名作。直到现在还是有资深动画迷喜爱，这一部作品也是后来的知名系列《世界名作剧场》的前身作品。

《海底两万哩》

动画OP

90年代GAINAX的名作。庵野秀明、樋口真嗣执导，贞本义行担任人设的作品，是许多御宅族心中的经典之作。

大川久男担任剧本的作品都暗藏一些特殊讯息在里面，一旦成人之后再重看一遍通常会有新的感受与体悟，是一位相当有才华的剧作家。

《海底两万哩》是在刺客猫小时候带来无限感动的好作品，在此祝这位老剧作家一路好走。
