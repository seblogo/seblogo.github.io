身份证号码中神秘的X到底是啥意思？

发表于 2020-10-06

大部分中国人的身份证都是由18位数字组成，不过也有个别的尾号为字母X，无限福利站长也是“X”系列，甚至过去给我带来不少困扰，那么这个神秘的X到底是什么意思？

对此好奇的网友不在多数，有人说是当年办户口的人打错了，也有说是抱养来的才打X，还有脑洞比较大的网友称，身份证带X的人其实是国家分布在基层的X特工！

好了，别瞎猜了。央视新闻微博日前已普及了这个冷知识：身份证的最后一位是根据前17位算出的校验码。如尾号是0-9，不会有X，如是10，则用X代替。这能保证身份证号不多位。

网友趣评“神秘字母X”：

我朋友给我讲过一个她干过最二逼的事。她以前不明白身份证末尾有X，然后有次她填表，偷瞄到别人身份证号末尾的X，心想，填个身份证还这么神秘，然后她就把自己的身份证号后四位都填成了XXXX。

刚上大学那会，学姐说饭卡密码是身份证号后六位，当我在超市花钱超了后，看着只要十个数字的密码机，于是默默的把东西又给退了回去。

一直都以为自己是散落人世的X战警。

身份证的各位数字代表的含义：

1、前1、2位数字表示：所在省份的代码；

2、前3、4位数字表示：所在城市的代码；

3、第5、6位数字表示：所在区县的代码；

4、第7-14位数字表示：出生年、月、日；

5、第15、16位数字表示：所在地的派出所的代码；

6、第17位数字表示性别：奇数表示男性，偶数表示女性；

7、第18位数字是校验码：也有的说是个人信息码，普通是随计算机的随机产生，用来检验身份证的正确性。校验码可以是0-9的数字，有时也用X表示。

动物界十大谣传趣闻

发表于 2020-10-06

很多人对动物先入为主的成见，导致了一些约定俗成的错误。比如兔子还吃胡萝卜，蜜蜂蜇人一次就要死，鸵鸟害怕时会把头埋到沙子里等等。实践状况是，这些看似最为人所熟知的“常识”都是没有根据的谬传。下边就让我们来看看，动物世界中，还有哪些你此前了解错误的趣闻。

旅鼠爱自杀

旅鼠这个名字还被用做形容不经大脑就效仿别人的人，由于旅鼠被以为经常不自觉地就会成群跳下悬崖。这全都“归功”于迪士尼1958年的一部叫做《白色荒野》的影片，它还获得了奥斯卡最佳纪录片奖。影片中，成群的旅鼠慷慨赴死。不过疑难就是，整个事情都是迪士尼一手策划，几乎是把旅鼠赶进了河里。

蜂只能蛰你一次

只要蜜蜂才只能蛰你一次，而且蜜蜂只是蜂类中很少的一部分。蜜蜂蛰完你会死是由于它们的刺尾带有倒钩，会埋入被攻击者，所以当它飞走时就相当于把自己扯成两半。至于其他品种的蜂，它们的刺相对光滑，所以蛰来蛰去也不会有什么疑难。

兔子爱吃胡萝卜

兔子是会吃胡萝卜，但是养过兔子的人都明白它们更爱吃绿叶蔬菜。错误的来源自然是动画片《兔八哥》了，但其实兔八哥一开始是在模拟电影《一夜风流》里的场景。

大象爱吃花生

跟上一条相似，这个错误也源于以前的动画片。但其实大象不仅不吃花生，甚至不喜欢它们。大象还是更喜欢干草、其他谷物和蔬菜水果。

我们每年会在睡梦中吞掉100只蜘蛛

这个数字也可能是50200，但没一个是对的，可能你终身中也就只能吞下去一两只。蜘蛛根本没理由会跑到你嘴旁边。这个“事实”要追溯到以前的万维网，它是想证明人们在网上看到什么都会随便置信。

黑豹是一个单独的品种

黑豹是很美丽，但它在动物界里不是一个单独的品种，它是豹属（基本包括一切大型猫科动物）下的一个亚类，而且是得了黑变病的。基本上它就是南美的捷豹和亚非的金钱豹。

鸵鸟害怕时会把头埋到沙子里

这要是真的就搞笑了，但鸵鸟还没被观察到有相似举动（除了在动画里）。鸵鸟遇到风险时跟别的动物一样，要么打架要么奔走，而且两样表现都不差。人家的速度能接近40迈（64公里/小时）呢。假如开打了，它有又大又锋利的爪，足够扳倒一头狮子。作为9英尺高300磅重的家伙，鸵鸟算是最接近迅猛龙的了。

北极熊捕猎时捂鼻子

异样，这要是真的也是够萌的。你以为它全身上下就鼻子是黑的，所以就要像忍者举动似的用爪子把自己隐藏在白茫茫的世界里吗？人们对北极熊的自然习性进行了上千小时的观察，也没发现它们捕猎的时分用这招。

奶牛不会下楼梯

这条错误是说你可以领着一头奶牛上楼梯，然后，就回不来了。还说是由于它们膝盖弯曲方式的疑难。这不完全对，奶牛下楼梯的能耐的确要看楼梯的高矮，但人家本来也不是为楼梯而生的，人家要楼梯干嘛。不过只要给它们工夫练习，奶牛还是可以好好下楼梯的。

大象鼻子像吸管

继续怪罪到动画片身上好了。大象的鼻子的确是自然界最独特的鼻子构造之一，但也仅此而已。大象鼻子又长又灵敏，可以用来抓东西，但首要义务当然还是呼吸。而大象绝不会做的一件事就是喝水的时分把鼻子当吸管用。假如看起来像，它们也只是用鼻子吸了水再送到嘴里去，并不是直接吸了就喝掉。

“当然了，正在给你静脉注射麻药。”这助手分明是主刀医生的帮凶。

“我手臂疼，太疼了，太疼了，受不了。”

“那怎样办呢，忍一下吧，打麻药哪有不疼的。”

我的手机呢，妈蛋！手也被绑了。

“医生，疼……”

2.

“柠檬，柠檬，醒醒……，喂，柠檬……”

迷糊中看到是带我去手术室的小护士，还有手术医生，还有李医生。

“你刚下手术台，下边伤口渗血，给你做一个减轻疼痛的仪器医治，60分钟，400块钱，你朋友在吗，需求去交一下费用。”护士的声响。

“能不做吗，我自己来的”，脑子混沌不清。

“最好要做，这是利于你术后恢复的，下边还在渗血，用手捂着，别拿开，等医治结束了你再去补交费用吧”。像是有人在梦里跟我说话。

“哦，好。”我迷糊着，于是背下边多了一个像按摩椅的东西，嗡嗡震动。

“腿打开，千万不要合，用手捂着，别松开。”主刀医生再次叮嘱。随后，人散。

手机忽然响了，是好朋友。“干嘛呢，打两个电话都没接。”

“哦，发烧了，在医院输液，手机调静音了，没留意。”她一听连问严重吗，在哪家医院，有人陪没，要过来看我。神志一下子清醒，赶紧说马上结束了，婉拒好意。挂了电话，内心惊险又暖和。

这是专门供妇科术前术后运用的输液室，严惩亮堂，只要我和护士，下午五点半，我侧过头，看到夕阳挂在窗子上，像是对我拥抱，顿然忘却了麻药过后的疼痛。

3.

至此，心病仿佛去了一半，躺在床上，无法得知整形做的如何，总归是要比术前好吧。

“医治结束了，我去餐厅给你拿点儿吃的，手术后体虚，需求补充养分。”护士一边整理仪器，一边微笑着对我说。我多么喜欢这样的敌对和贴心，这还是那个催我缴费的护士么？

片刻，她端了牛奶，豆浆，鸡蛋，还有小面包放我床头。体虚，也没胃口，只喝了半杯热牛奶。之后，下床去找李医生，她会交代术后护理的事项。

不明白是心思作用还是医治仪的作用，并没有很激烈的疼痛感，只觉得下边湿粘，酸胀。李医生说是正常景象，又给我换了吸血海绵条和三层医用纱布，手术结束后垫的纱布已经湿透，混着血色和黄色的液体。李医生让补交了医治费，并吩咐尽量不要走路，减少摩擦，有疑难及时联络她，第二天要去医院做术后护理。

4.

真的是太心急，打车到家后，不安又好奇地小心翼翼拿开纱布，照镜子看了一下手术效果，伤口渗着血珠，晶亮闪闪，肿得完全看不清形状，赶紧垫好纱布躺下。过一会儿，分明感觉有液体从屁股下边流淌，起身看，床上湿漉漉的鲜血一片，简直要吓死。我赶紧给李医生打电话，拨了十几次都是无人接听，海绵条和纱布被血浸透，我又在湿透的纱布下边加了一片卫生巾，趴床上一遍一遍打给李医生，她这根唯一的救命稻草也联络不上，着急又害怕，会不会大出血死掉？懊悔的不行，为什么要做这该死的手术？一定是那个红棉拖医生医术不行，随意缝了几针，还有李医生，明明问她要个人电话，说打名片上的电话就可以，你妹，一定是没有带手机回去，大骗子！边流泪边不怀好意地揣测着医院的每一个人，大概半小时，发现不再哗哗流血，依然有血珠渗出，那一晚，内心复杂到难以入睡。

第二天，李医生对没有听到电话进行了辩解（说没有收到来电）和道歉。接下去进行了延续四天的术后护理，每一次都是微米光照射40分钟，输液三瓶消炎。

5.

术后护理时期，针对手术的一些细节疑难一直心存芥蒂，再次和李医生沟通，过程很不愉快。今天是术后的第13天，伤口已经康复，没有任何举动阻碍和不舒适，深感是幸运之神的眷顾，自始至终不情愿承认完全受益于医院的医治。在医生眼里一个非常普通的小手术，依然带给我了极为丰富的心历路程，特别是自我看法和自我保护方面。希望上述如此坦诚的描绘和接下去继续坦诚的内容不会被和谐掉。

心病成魔

由于自小对身体发育的看法粗枝大叶和后知后觉，我一直觉得我的身体发育状态是什么样的，别人也就是什么样的，就像有一天同窗很惊奇的发现我眉毛会动，耳朵会动，手指不用掰就会响，我无辜得像个傻子，“难道你们不是这样子的么？”所以对于私处如此隐秘羞怯的地方，我更没感觉到有任何疑异之处，我以为全天下姑娘都是一样的，很傻很天真的以为了很多年。

直到去年一个偶尔的机会理解到自己的状况属于YC发育畸形。残酷的真相直接让我麻爪。今后结了婚老公会不会怀疑啊，会不会误解啊，会不会厌弃啊，我还是纯良妹纸啊……于是在各种碎片工夫里总是不自觉的想到这些，尤其是洗澡的时分，这直接导致我对自己身体该处的极度在意和隐隐的自卑，完全不明白该怎样接受自己的不完美，就这么轻而易举的患了传说中的心病。更是由于私处这么特殊的身体部位，羞于向家人朋友诉说，只要依赖网络高密度搜集相关资料。网络信息琐碎庞杂，负面偏多，内心更加恐慌，压力剧增，素日的自信由于这个被消磨殆尽。

网上有很多失败的案例，各种苦楚，各种后悔充斥屏幕，看的我小心脏丝丝发凉，但又心存侥幸，鬼使神差的查找本地相关医院，在多达近3页的百度广告里，挑了一家自称据有”省级资质”的医院，而不是XX妇科专业医院这么分明的牌子。打电话询问了各种事项并表达了顾虑，电话里的姑娘晓之以理动之以情极力让我去趟医院，说让主治医师检查一番根据实践状况才可订手术方案，并免预定，简直是盛情难却，渴望改变是非常可怕的原动力，于是有了开篇的那一幕。

沟通不畅

躺在手术台上了，主刀医生还在问我各种疑难，这些本该是主治医师在术前对我理解的，尤其在麻醉的疑难上让我一直心存芥蒂。手术前，李医生不止一次告诉我只是一个非常小的整形手术，半小时就结束了，打麻药，微创无痛。由于她总是在强调渺小手术，也是为了安抚我，所以我一直以为是局部麻醉，我没问，她也没提，躺在手术台手脚全缚后被告知是全麻，第一反应便是我不会醒不来吧，由于无知而恐惧，还有被隐瞒的愤怒，直到闭上眼睛前一秒，身心都处于一种极度恐惧无助的状态。

再者关于费用沟通。李医生直接引荐了最贵的方案费用是3000+，韩国微创整形技术，无痛便捷，不拆线，为了安全保险我接受了她的建议，术前检查费用且不说，下手术台后让做减痛按摩医治，术后每天术后每天微米光+输液500元，延续四天，这些术后护理费用她只字未提，我由于从网络上理解到术后用高锰酸钾自己清洗就可以了，所以也没问，后来问她为什么不告诉我这些，作为病人有权明白的，不能由于我不懂你就不告诉我，李医生很无辜的说手术后都是要护理的呀，你这种手术都是全麻的呀，还用说嘛，我每天这么多病人，不能每一个人都跟她说一遍吧，你就是太心细太敏感。心境非常坏，我言辞也不敌对，并提了第一次来医院咨询时章医生跟我讲的手术方案和她的完全不一样，当然章医生更不靠谱，告诉我她亲自来给我做，就在她坐诊的帘子后面，下手术台就可以照常走路下班，在她口中就像手指划一道贴个创可贴这么简单似的，万分庆幸第一次没有冲动，李医生说我诋毁她的好朋友，她不可能说这么没水准的话，手术也不可能在诊室里做……，把我赶出了诊室。时期由于延伸护理天数聊的非常不愉快，由于她的不敌对我就觉得她是忽悠我高额护理，手术全部下去一共花了近7000元，大大超出了她说的3000多一点儿。

一点感触

其实我手术前的状况没有走路摩擦或者运动夹痛的不适，并没有网上说的那么严重，所以这么多年我没有任何心思负担。所谓多知多风险，又羞恐与家人朋友沟通咨询，导致内心压力过大，当时急于改变的心态往往会忽略与利益共存的风险，或者自我暗示降低风险，很庆幸手术没有任何疑难，恢复良好，到如今依然没有一个亲友明白我做了私处整形手术，今后也不会告诉他们，我一个人下班去输液，一个人躺在手术台以最暴露的姿态接受术后护理，一个人半夜害怕到哭出声来……

而我要告诉锦友的是，经历过后回头看，身体得以医治，我并没有那么开心兴奋。些许后怕些许庆幸，遇到疑难及时和最密切的人沟通非常重要，压力是用来缓解的而不是积攒的，爱自己，客观感性地看待自己的不完美，而这不为人知的心里路程让我内心越来越宁静，这才是最好的。
