文章：客官，这碗鸡汤有毒

发表于 2020-10-06

文/梁佩湘

客官，这碗鸡汤有毒，别喝！

我是一个业余到不能再业余的写手，从高中开始在线上码字发文，看着自己写过的大大小小文章，想想都激情澎湃了一把。然而，我却得了一种 “这辈子大概只能煮煮鸡汤”的职业病。每次写文章，最后文风都带着浓浓的鸡汤味，每次都干着努力说服别人的活。这种感觉就像是食堂大叔本来是想做狮子头，到最后却做出了一坨坨差强人意的四喜丸子。

鸡汤文语言温和，充满正能量，一看就有一种软妹子在对着你叫“萌萌站起来”的即视感。干了这碗鸡汤，喝汤者也就头脑发热一下，觉得自己不得了，怎么可以虚度韶华，必需得努力奋斗啊！然后就是一轮马不停蹄、斗志昂扬地给自己写计划，譬如要在多少天内看完哪本书，今天要跑多少圈操场，要背多少个的单词。但等鸡汤药效一过，头脑清明，大家都会发现，大道理都懂，小情绪难以自控，不能解决的问题还在晾着，该拖延的工作还等着deadline的到来。这碗鸡汤，喝了真的会醉。

活了二十小半生，我们其实都已经过了耳听鸡汤的年纪，都知道鸡汤即鸡血，只能精神上亢奋一段时间。人一旦冷静下来，发现问题依旧摆在那里还没解决，不来不去。

我不得不告诉你，尽管我在煮鸡汤上功力了得，但是鸡汤一直都解救不了我，每次写完文章，把一群人弄得热血沸腾后，自己反而内心down得不得了。我其实一点都不积极。因为不知道明天和意外哪个先到来，所以我一直都做着“死前不做会后悔”的清单。

嗯？我已经做了什么？在二十岁之前，我已经争取认识了好多牛逼的人，然后跟着他们体验了好多事情，比如像现在这样让你读着我的文章、比如让你点开微博广场能看到我做的图、比如让你在某电台看到我的痕迹、比如能让你在搜索引擎上一输入我名字就能找到我。当然，文章可能是你偶尔打开的，图做得可能也并不符合你的审美标准，或许我自认为的了不起其实在你眼里也没有多了不起。你说，这是时间问题，过两年你也会做到。我翻翻白眼，好歹这些事情我在做了，你能活到那时候再说吧。

这些事情的本身的确一点都不牛逼，我以前也觉得没有什么大不了的，但是在耳听过那么多大大小小的意外死亡后，我就真的觉得，还好，我今天还活着，而且这些事情我都做了。至少，“死前不做会非常遗憾”的事情清单又少了一些。靠着20岁的年龄，40岁的身体，将近80岁的心，却有一颗13岁说干就干的行动力，能走到这一步，我已经知足了。

什么是生活，大抵就是生下来，然后如何活下去的归论吧。像我这样的文青，在最需要鸡汤的那段时间，就是读着卢思浩念着李阳的疯狂英语一路热血奔腾过来的。现在还有很多人喝着卢思浩的大众鸡汤，而我们都老了，老到自身的消化系统再也消化不了鸡汤。该难受的时候还是继续在难受。不能解决的问题即便你把鸡汤煮了七七四十九天，把一锅熬成一碗喝也解决不了。

大家都是成年人了，还喝什么鸡汤？下次心烦意燥的时候，不如考虑温一碗烈酒灌衷肠。一醉，一醒。第二天，继续生活。毕竟，生活还在继续。

1 鬼乡 （2016）

导演：赵正莱

主演：姜荷娜 / 崔莉

《鬼乡》大概是近期最火的韩国电影。

虽然还没办法看，但相信你已经多多少少都耳闻了这部电影。

《鬼乡》讲的是关于一个慰安妇的故事，它以一位真实且仍然在世的日本慰安妇受害者为原型，把当时那段惨绝人寰的历史还原给你看。

你能在这部电影里看到什么？

正值花季雨季的15岁少女被强征为慰安妇，失去了最宝贵的青春。

她们在真正意义上成了我们生活里常常拿来看玩笑的“公共汽车”，谁想上就能上，连票都不用买。

如果她们有一丝想要反抗和不听从的举动，就会惹来一阵毒打。

万一患上了传染病，唯一的结局就是被烧死。

如果你有关注这部电影，你或许知道这部电影的导演拿着剧本四处寻觅投资方，却没有人愿意出钱。

最后只得在网上发动众筹，才成功拍下了这部电影。

电影上映之后，首周的观影人次就突破百万，引起韩国空前的反响。

人权与反战让人们深思。

慰安妇问题再一次成为社会大众的舆论焦点。

而她们，也确实需要人们的关怀。

即使当年在军队的刀刃下死里逃生，之后的生活也会引来大众的鄙夷和排斥。

就连慰安妇之子都像是社会毒瘤一般，不受待见。

而他们内心的艰苦，有人关心吗？

《鬼乡》能把这种带有歧视的观念扭转到什么程度？不得而知。

但是这部电影却能让观众血淋淋地看着慰安妇过去承受的折磨。

当下一次他们再想向曾经的慰安妇输出偏见的时候，话到嘴边，或许会嘴软。

无论如何，在这条平等的道路上，筹备14年的《鬼乡》一定会成为强而有力的助推器。

2 恐怖直播 （2013）

导演：金秉祐

主演：河正宇 / 李璟荣 / 全慧珍 / 李大为

真人真事改编的电影本身就自带震慑力，因为你总能自己脑补这竟然是真事儿！

但不是真人真事改编的电影同样也足以改变你，《恐怖直播》就是个佳例。

由河正宇饰演的广播主持在节目中接到了一通神秘的电话。

一开始，对方先是不断抱怨社会对他的不公，而后竟然转变成扬言要炸掉汉江大桥，可怕的是他也确实这么做了。

这位广播主持决定抓住这次惊天的独家新闻，为自己的主持事业拼搏一次。

他首先找了电台长官，保障自己今后能变成露脸的主持人，并让自己独家直播。

长官认为这是个冲高收视率的良机，他也答应了这个请求。

然而，事态开始变得一发不可收拾。

河正宇饰演的主播开始全程直播这次通话，但是电话连线的另一头就是连环爆炸案的犯罪嫌疑人。

这样的事情当然能引来各方关注，甚至是青瓦台。

《恐怖直播》是一部很勇敢的电影，它不但黑了一次电视界的惊人暗箱，就连政府甚至是总统和国家都不放过。

所有人都是自私自私的，毫无例外。

要换作国产电影，可能都还来不及演到政府出面。

开头10分钟就会被广电给禁了。

收视效益、个人恩怨、国家安全、政府颜面，从社会底层到国家高层，此时因为一通电话全牵扯到了一起。

电视台的领导为了收视率打算牺牲人质和下属；一个不服凶嫌的警察署长惨死在凶嫌的操纵之下；给直播间说一套却在媒体前做一套的青瓦台；从一开始就没有打算让这位主播活命的政府……

所有人都在直播的过程中盘算着自己的利益，而河正宇饰演的这位主播呢？

活生生成了这场利益博弈的牺牲品，在主播台上无法动弹，任人宰割。

为什么会产生这种悲剧？

传媒界缺乏职业道德，为了利益不择手段；政府面对问题互相推诿，态度强硬。

就连最基本的社会保障体系都丢失了，导致炸弹犯的出现。

这部电影无畏剧透，因为这种对人性劣根的阐述，需要你亲身观影才能体会。

看完以后希望你不会像我一样怀疑世界：这个社会还有真正的勇敢和良知吗？

3 那家伙的声音 （2007）

导演：朴镇彪

主演：薛景求 / 姜东元 / 金南珠 / 金英哲

这部电影根据上世纪90年代韩国的一起真实社会案件改编，令人发指的李亨孝绑架案。

1991年，一位名为李亨孝的9岁男孩被拐，犯人开口就要了1亿韩元的赎金，在之后还不断提高。男孩的父母当然报警，警方前后共派出15万人全力追查。

但这却是一次高智商犯罪，犯人用各种手段逃脱了警方的各种追查。最后，警方甚至连犯人的长相都不知道，唯一的线索是犯人那阴郁且冰冷的声音。

最终，犯人拿到了2亿韩元的赎金。但是在事发44天之后，男孩的尸体在汉江边被发现，他在事发第2天就已经身亡。

然而犯人却在之后的40多天里，不断折磨着男孩的父母，同时折磨着观众。

在电影里，饰演这名犯人的是姜东元，他和黄政民一起主演了前阵子很火的《检察官外传》。

与真实案件一样，姜东元始终没有露正脸，但他却把犯人那种散着寒意的声音模仿到了极致，就连台词都几乎还原。

《那家伙的声音》上映时，事件已经过了16年。15年的追诉期已过，警方撤回所有追查警力，然而凶手依然逍遥法外。

这部电影会改变什么？

导演在征得男孩父母的同意后，把事件改编成了电影，并在结尾播放了当时犯人的录音和可能的肖像素描。

他将这起骇人听闻的撕票案公布于众，让社会大众亲身体会一次这种惨无人道的犯罪所带来的身心折磨。

让社会能提起警惕，杜绝事件再次重演。

也希望能借由社会大众的力量，将这个冷血的杀手彻底绳之于法。

4 孩子们 （2011）

导演：李圭满

主演：朴勇宇 / 柳承龙 / 成东日 / 朱进模

《孩子们》同样根据真实社会案件改编，这起重大的刑事案件同样发生在1991年，被称为“青蛙少年失踪案”。

这起案件和上文的“李亨孝绑架案”都是韩国社会的三大未解悬案之一，还有一起案件同样也被改编成了电影，那就是被很多人奉为韩影影史最佳的《杀人回忆》。

1991年的春天，5名少年外出抓捕青蛙而后失踪。

事发后，全韩共动员了30万名警力，印发总计2亿张传单，在香烟盒上张贴寻人启事，时任韩国总统卢泰愚给予特别关切，中小学发起“寻找青蛙少年”的活动。

但最终都一无所获。

直至事发11年以后，5名少年的遗骸才在附近的山上被找到。

事隔多年，除了能确认是被钝器所伤的他杀，其余线索全无。

《孩子们》就是根据这起社会案件改编。

在电影里，导演选择以一名电视制作人的视野为叙事角度，并把焦点放在了事发后的案件追查上。

等着猛料的媒体、5名少年各自的家人还有社会的大众。

对于这起悲剧，大家都怀揣着各自的心思，人间百态。

《孩子们》起到了什么影响？

这起案件早就过了15年的追诉期，但是凶手仍然逍遥法外。

这在社会上引起了巨大舆论反响，少年们的死因未定，真凶也还是谜，人们都认为这起事件不应该因为过了追诉期而停止追查。

为此，社会大众集聚发声。

提出了希望能追加反人道犯罪追诉期的要求，避免再出现类似“青蛙少年”这样的悲剧，让真正的罪犯能够彻底伏法，让社会能有公道。

接下来要介绍的4部电影不止能给观众起到警醒作用，它们还影响了观众的实际行动。

这些电影甚至促成了案件重审，促成了法律修订，在真正意义上改变了现实。

5 梨泰院杀人事件 （2009）

导演：洪基善

主演：张根硕 / 郑镇荣 / 吴达洙 / 宋仲基

梨泰院现在是韩国首尔的一个著名商圈，但这里曾经发生了一起让人后脊发凉的案件。

在90年代，梨泰院还是美国驻军和军眷的居住地，有着大量的红灯区和韩美混血儿。

案件发生在梨泰院的一家汉堡店内，一名韩国大学生莫名惨死在店内的厕所里。

警方在后来将犯罪嫌疑锁定在两名现场目击者身上，一名是美军家属，一名是韩裔美国人。

经过一次次审讯，警方几乎确认了犯人就是这两名青年，他们联手用凶器猛击被害大学生，导致其伤势过重而身亡。

在一审和二审时分别被判处无期徒刑和20年有期徒刑，但是让人惊掉下巴的是，这两名犯罪嫌疑人最后因为证据不足而被无罪释放，法官推翻先前全部判决。

你没看错，无罪释放。

《梨泰院杀人事件》就是根据这起案件改编。

顺便一提，饰演被害韩国大学生的，就是你们的宋仲基欧巴！

1999年，犯罪嫌疑人在被释放后立即逃出韩国，引起了轩然大波。

为了还原事件，导演用了整整四年的时间，采访了40多位当时对这起案件跟踪报道的记者。

试图把案件最真实的状态展现给观众。

这部电影甚至因为犯罪场面过于血腥而无法过审，剧组为此修改了4次画面才得以上映。

但案件最真实的状态就是让犯人逍遥法外，直到《梨泰院杀人事件》上映之后。

虽然在这部电影里没有抓到真凶，但在影片上映之后，这起案件引起社会大众的广泛讨论。

人们纷纷发声表示不公，他们认为凶嫌明明已经确认，竟然不抓？！

最酸爽的是，《梨泰院杀人事件》不仅让这起案件重新获得关注。

更在群众的帮助下成功促使案件重审！

电影上映之后不久，韩国官方就正式对当年潜逃出境的犯人提出引渡申请。

直到去年9月底，距离1997年已经过了16年，这名美军家属终于被遣返回韩国。

真凶被捕，《梨泰院杀人事件》简直为韩国和社会大众正了一次法！

6 素媛 （2013）

导演：李濬益

主演：薛景求 / 严志媛 / 李来 / 金海淑

一论起改变了韩国的电影，我相信你的脑袋里一定会浮出《素媛》这部电影。

豆瓣9.1分，杀入豆瓣电影Top250，2013年的青龙奖最佳影片。

即使单论影片本身，《素媛》都有十足的观赏性。

更何况这部电影同样根据真实案件改编！

《素媛》改编自一起幼童性侵案件：2008年12月，韩国安山市檀园区，一位名为赵斗淳的流浪汉在路上拦下了一名正要去上学的8岁女童，并对她实施暴力性侵。

赵斗淳的性侵手段非常残忍，为了毁灭证据，他在完事之后甚至用木棍硬捣女童下体。

累积的种种伤害导致她下体严重破裂，身上多处重伤，致使终身残疾，必须永久依靠排便袋。

但是《素媛》展示的不是这起案件的经过。

而是这名女童在遭受性侵之后，她如何一步步抵抗心理阴影，她的父母和亲朋又如何一步步接受并善待这个事实。

让人心寒的事是，虽然证据确凿，赵斗淳不仅没有认罪，还千方百计找理由推脱这些证据。

他甚至以自己当时是病理性醉酒为由，认为自己丧失了自主能力，企图量刑。

法官也听信了这个理由。

在悔改态度不佳、犯罪行为铁证如山的情况下，赵斗淳只被判处12年有期徒刑。

《素媛》上映之后，这起案件再次引起社会的关注，引爆全民讨论。

几乎整个社会都怒了。

案件被披露后，在社会舆论的强压之下，韩国法院为此修改了法律。

不仅加强了针对儿童性犯罪的量刑标准，还将对儿童实施性暴力犯罪诉讼的实效延长至受害儿童成年为止。

同时，提出了加重性暴力犯罪的刑期上限。

除此之外，还否定了“病理性醉酒”，这部电影甚至推动了化学阉割的通过。

《素媛》的轰动，不仅能让幼童在更大的保护伞下成长，让罪犯受到更大更重的刑罚，还能让韩国社会更加重视对儿童的性暴力犯罪。

7 熔炉 （2011）

导演：黄东赫

主演：孔侑 / 郑有美 / 金志英 / 金贤秀

一论起改变了韩国的电影，除了《素媛》，最让人印象深刻的就是《熔炉》了。

豆瓣9.1分，杀入豆瓣电影Top250的前50。

这部电影同样以一起真实的性暴力事件改编，这个事件发生在2005年韩国广州的一所聋哑特殊学校里。

《熔炉》的前半段讲述的是事发的过程，新来的美术老师在学校发现校方领导性侵年幼学生的现象。

他在保证自己的教师职位和人性良知之间选择了后者，开始阻止老师的行为。

到了后半段，这名美术老师开始与校方领导对簿公堂，而领导们也开始用各种手段为自己洗脱嫌疑，甚至私下用金钱贿赂学生家属，达成和解。

现实生活中的这起案件，同样如此。

尽管你在电影里能看到校长在厕所猥亵女童、在办公室播着毛片强奸学生。

还有男老师在男学生的家里强奸他和他的弟弟，不听话就是毒打。

这些触目惊心的画面虐到让你窒息。

但是这些罪犯最后却只获得了几年甚至几个月的有期徒刑，还带缓刑的那种。

当男童的奶奶决定和对方达成金钱和解的时候；

当男童知道自己已经无法再为此做任何努力的时候；

这样的画面像被抓着喉口一样让人喘不过气。

也因此，《熔炉》改变了法律，它彻底改变了韩国。

《熔炉》上映后，网上出现了百万人签名重启调查的请愿活动。

光州警方重新调查，并在大众舆论力量的支持之下，法律得到修改。

《性暴力犯罪处罚特别法部分修订法律案》（又名“熔炉法”）规定，对残障人及不满13岁的儿童的性暴力犯罪不受时效限制；对残障人性暴力犯罪的，删除了“不能反抗”的构成要件；强奸犯罪的处于七年以上或无期徒刑，强制猥亵犯罪的处于五年以上有期徒刑，且不得宣告缓刑；从事残障人保护及公益事业的人员，对残障人实施性暴力犯罪的，在量刑标准的基础上加重二分之一刑期以严惩。

《熔炉》没有像《素媛》那样煽情动人，但是却比《素媛》虐心百倍。

影片中那一幕幕让人瞪大双眼的画面，可能短时间都忘不掉了了。

8 辩护人 （2013）

导演：杨宇硕

主演：宋康昊 / 吴达洙 / 金英爱 / 郭度沅

《辩护人》以真实发生过的社会事件“釜林事件”为蓝本进行改编，核心人物是后来成为韩国总统的卢武铉。

1981年左右，韩国政府对釜山地区的大学生活动家进行大规模的拘留审讯。

他们以违反《国安法》为由，随意施虐让其认罪，栽赃大学生。

当时还只是普通律师的卢武铉奋不顾身地替被害学生进行人权辩护。

在《辩护人》中，以卢武铉为原型的主角由宋康昊饰演。

《辩护人》还原了釜林事件的始末，时任总统全斗焕的独裁政府被搬到阳光底下，再一次让观众审视，整个法制体系的巨大问题一目了然。

揭领导人之短以及把黑历史当题材的电影，在我国全都成了禁片，无一例外……

然而，《辩护人》改变了什么？

《辩护人》上映之后，釜林事件再一次跃进大众的视野里，社会大众再一次提起了当年的这个热门事件。

就像《熔炉》或者《素媛》一样，随着电影的火爆，这起事件的社会关注度日益增高，舆论漫天飞舞。

终于在电影上映后的隔年，“釜林事件”进行复审宣判。

33年前被判有罪的其中5名被告，最后被宣布无罪。

当年的莫大冤屈在多年之后终于洗白，这就像是一次大补的鸡汤正能量让韩国社会又强壮了一次。

社会必将是公正的，社会必将是良知的，但是当社会出现问题，当国家出现问题，那就是需要社会大众勇敢发声的时候了。

电影能改变国家吗？当然不能。

但是电影能改变社会群众，这些优秀的文化内容潜移默化地影响着社会大众，当社会大众的公正和良知被唤起，直到未来的某一天，他们将作为最强大的力量改变社会和国家。

就像《辩护人》的导演杨宇硕曾说过的：“卡夫卡说过这么一句话，来形容文学和现实的关系——冬天用斧头打湖面上很厚的冰，是打不破的，可是会惊醒冰面下的鱼，但最后融解这个冰，一定是春天，不是斧头。电影，就是这个惊醒老百姓的斧头。”
