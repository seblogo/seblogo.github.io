好莱坞电影《七龙珠》编剧时隔7年向粉丝致歉

发表于 2020-10-06

2009年上的好莱坞电影《七龙珠（英文名：DRAGONBALL:EVOLUTION）》从上映开始就饱受批评，并且引起众多龙珠粉丝不满。惨剧过去将近7年，近日该电影的编剧本·兰姆赛终于正式站出来向龙珠粉丝道歉。

《七龙珠》是香港导演黄毅瑜执导的一部日本动漫改编好莱坞电影，由知名香港影星周星驰担任制片人，本·兰姆赛负责编剧，周润发和加拿大演员贾斯汀·查特文担任主演。2009年3月14日在中国内地上映，2009年4月6日在美国上映。该片虽然打斗特效华丽，但失去了原著的精神，而成为一部典型的美式青春剧，不但让死忠龙珠迷非常不满，连原著鸟山明老师也感到失望。

近日一位名叫德里克·帕杜拉的作家采访了该电影的编剧本·兰姆赛，兰姆赛本人也通过这次采访检讨了自己犯的错误，并以文字的形式发表了公开道歉，全文如下：

“我知道这一天会到来的。《七龙珠》对作为创作者的我来说称得上是人生最大的污点。我也被冠上了世界上评价最差的剧本家的称号，这让我的心仿佛被撕裂了。直到现在我还能收到来自世界各个地区的指责我的邮件，我感到非常心痛。

这些年我也想过转嫁责任，但剧本里的文字全部都是我写的。让许多粉丝感到失望全部责任都在我身上。我虽然曾经竭尽全力想做好，但还是弄砸了。

我不是作为《龙珠》的粉丝，而是作为一个只为追求巨额报酬的商人来写剧本。我知道勉强去做自己没有热情的东西是做不出好作品的，甚至还可能会沦为彻底的垃圾。

责任不在别人，都是我的错。我非常理解粉丝们看到自己心爱和期待的作品被做成这样的糟糕的东西的心情，因为我也是其他作品的粉丝。

我想对《龙珠》的各位粉丝表示深深地道歉。”

好莱坞人才为自己的作品道歉是非常罕见的事情，而且道歉中兰姆赛本人更是直白的承认是为了钱而写的剧本，更是闻所未闻，他的诚意和诚实也得到了粉丝普遍的认可，有粉丝表示：“我能够接受这个道歉，虽说已经过去7年，但能这样有诚意地道歉还是很需要的勇气的。希望这个失败的作品能给他带来教训，以后能做出真正厉害的作品。”也有粉丝觉得从动机来说他不是特例：“他说自己是个只为追求巨额报酬的商人这一点，在动画业界、游戏业界、声优业界……等各种业界都存在同样的问题。好好做自己有爱的作品，粉丝的眼睛是雪亮的。”还有粉丝认为从电影制作方面来说，锅也不应该他一个人背：“责任应该不只在剧本家一个人身上…毕竟电影制作涉及到的人非常多，我认为他没有必要自己一个人承担所有的过错。”

众筹动画《像风一样》排定上映日程

发表于 2020-10-06

由千叶彻弥原作改编的短篇动画《像风一样（原名：風のように）》已达成众筹目标，正式确定于7月9日在东京下北泽一家影院独家上映，另外5月7日还将举行完成披露谈话活动。

《像风一样》是1969年在讲谈社发行的漫画杂志《少女Friend》上刊登的单篇完结的漫画作品。这部作品以在人们普遍开始关注环境问题之前就探讨了人类与自然该如何共生的问题而闻名。

《像风一样》的动画是从2016年2月开始众筹，当时的目标是众筹到200万日元，而实际筹到了300万日元。动画由Ekura Animal制作，监督是本多敏行，作画监督为野口征恒，原画则是木村圭市郎、大桥学、增田敏彦、田边修等实力派制作人员。主角则由配过《龙珠》、《四驱小子》等经典作品的野泽雅子配音。

5月7日将为参加众筹的支持们举办完成披露谈话活动，作者千叶彻弥、漫画家村上纪香、石川サブロウ、川三番地都将到场，监督本多敏行和作画监督野口征恒也将出席，歌手松本佳奈还将现场演唱动画的主题曲《恋风》。6月11日将在栃木県樱市举办发布会，作者千叶彻弥与漫画家森川让次也将出席。电影7月9日在东京下北泽的影院独家上映。

《黄金神威》获得漫画大奖后销量爆增

发表于 2020-10-06

由野田サトル原作的漫画作品《黄金神威》获得了今年的漫画大奖，据悉，该作品获奖后的2个月里销量翻了至少2倍。

在《黄金神威》获奖前的3月，已经出版了6卷单行本。全6卷单行本累计销量只有100万部，在获奖后该作于4月19日发行第7卷，截止目前，前7卷累计销量达220万部以上。另外，还有电子书市场的销量也非常乐观，有知情者表示：“目前给出的只是速报值，实际上销量应该可以翻3至4倍”。

漫画大奖与出版社为表彰旗下漫画而设的奖项不同，该奖的评选不能有出版业从业人员参与。而且能够获得这个大奖的漫画销量都会急速增加，因此每家出版社都会觊觎这个大奖。《罗马浴场》、《银之匙》等就是获得该奖后开始大卖的漫画作品，之后还被改编成动画和真人电影。相信获奖的《黄金神威》在迎来销量爆增后，离动画化也不会太远了。

《黄金神威》是由漫画家野田サトル创作的漫画作品，在2014年38号的《周刊少年JUMP》上开始连载。故事讲述的是为了实现在1904年日俄战争中死去好友寅次一掘千金的愿望，军人“不死的杉元”从东京搬到北海道，途中结识了阿依努女孩阿希莉，二人踏上寻找黄金的征途。另一方面，警察和第七师团也一边进行着追踪交换，一边追逐线索。2016年3月29日该漫画获得了2016年漫画大奖。

日本国宝级音乐家富田勋去世

发表于 2020-10-06

日本作曲家兼电子音乐先锋富田勋本月5日因慢性心力衰竭在东京一家医院内病逝，享年84岁。

富田勋出身于东京都，在应庆大学读书的时候开始真正尝试作曲。他是第一位成功把电子合成乐引进古典音乐的乐师，也是古典合成派的代表人物。他的音乐圆熟地将西洋管弦乐、人声、合成器以及日本传统乐器交融为壮大的乐声，创造出深邃神秘、奇幻玄妙的音效世界，耐人寻味，是日本电子合成器演奏大师，也是世界少有的电子音乐奇才。

去年83岁的富田勋第一次“回”中国举办音乐会。他小时候在中国生活，因为在天坛回音壁玩，发现声音的神奇，对他后来从事音乐创作有了启发意义。音乐会上富田勋不仅运用交响乐、合唱与电子音乐相融合，更通过全息投影技术将风靡国际的虚拟偶像歌手“初音未来”带到音乐会的现场。

5月5日富田勋倒在位于东京品川区的自家住宅中，随后被送往医院救治，东京时间下午3点前，因心力衰竭死亡，享年84岁。5月7日和8日亲属为他举办了追悼会。富田勋的儿子表示：“父亲非常期待在今年11月举办的音乐会，倒下前1小时候还在兴致勃勃地商量着音乐会的事。父亲的作品和意志不会逝去。请给位今后也要继续支持富田勋。”

让男性觉得可爱的日本女性名字

发表于 2020-10-06

当动漫迷在看日本动漫的时候，是否也会记下其中很多妹子的名字呢？而实际上，有些角色就因为名字好听而被大家牢记，这个现象在现实中也会出现。为此，近日就有日本媒体评选了“会让男性角色可爱的女性名字”。

■まい（MAI）

“不知为何，就觉得是可爱女孩子的名字。”（23岁/其他/其他）

“一直以来，叫这个名字的可爱女孩就非常多。”（31岁/信息·IT/技术）

“我遇到过的MAI都很可爱。”（28岁/学校·教育相关/事务系）

应该说MAI也是有着稳定人气的名字呢，而汉字往往是麻衣、真衣、舞等等，不过近年来也多了舞彩、真爱这样的汉字，显得非常丰富多彩。光是看着汉字，也会让人觉得可爱了吧？

■りさ（RISA）

“RISA。让人一种大人的感觉。”（32岁/运输·仓库/技术）

“RISA吧。或者RIE之类的，反正用RI开头的名字。”（33岁/小卖部/事务系）

“RISA。感觉是开朗美女的形象。”（32岁/学校·教育相关/专职）

RISA会给人可爱而成熟的感觉。不过叫这个名字的日本人似乎并不多，感觉更像是外国人的名字呢。如果写成汉字的话，应该是理沙和里咲吧。

■あやこ（AYAKO）

“AYAKO，叫这个名字的往往都很可爱。”（33岁/汽车相关/事务系）

“AYAKO。可爱的可能性很大。”（30岁/运输·仓库/技术职位）

“AYAKO。给人美女的印象。”（34岁/媒体/技术）

AYAKO一般会写成绫子之类的汉字。并且“子”给人一种古风女性的印象。不过在如今这个年代，古风反而更加受到欢迎呢。

■花名

“堇。给人楚楚动人的印象。”（37岁/机械/销售）

“樱。这是将日本的美进行概括的名字。”（35岁/金融/营业职位）

“桃。像是治愈系的名字。”（33岁/信息·IT/事务系）

现在也有不少父母给孩子直接使用花名呢。毕竟这些名字光看上去就很美丽了。

■总结

名字往往会给人不同的印象。光是看名字的假名或者汉字，就让人产生“美女”“知性”“土豪”等各种各样的想法了。那么你最喜欢的日本女性名字又是什么呢？
