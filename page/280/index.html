Comic Con《老外选出最差劲的5本漫画》最优秀漫画又有哪些呢？

发表于 2020-10-06

美国圣地牙哥宅宅盛典Comic Con今年也风光举办，今年也集结了外国漫画业界人士，讨论选出有在北美地区出版的日本漫画当中，他们觉得最优秀跟最差劲的作品，其实这些最差劲漫画～在我们亚洲读者眼里看来应该还好，可能真的是文化差异的问题吧…除了《老外选出最差劲的5本漫画》也一併让大家了解一下他们认为优秀的漫画有哪些作品唷！

最差劲作品

黑色五叶草 ブラッククローバー／作者：田畠裕基

《黑色五叶草》曾获「日本书店店员推荐漫画」以及「读者票选漫画大赏」，应该是很不错的新连载作品啊？但或许就因为太过王道，甚至在日本读者眼中有点「根本是火影忍者」的感觉，所以让评审觉得有点无趣吧？

朋友x怪兽 トモダチ×モンスター／作者：乾良彦

乍看名称好像描述很热血的友情故事，但这绝对不是一款适合小朋友看的怪兽战斗作品，虽然原作里是小学生们在战斗，但却有相当黑暗血腥的内容，怪物还真的有不少丑陋恐怖的怪物、断头断肢的画面，是实际在杀戮的战斗漫画…

千年の翼、百年の梦／作者：谷口ジロー

由《孤独的美食家》作者与罗浮宫合作的漫画，以罗浮宫为舞台交织出不可思议的旅程。说真的～评审们为什幺选出这些作品的原因还真不得而知（或许有去参加的朋友会在座谈会上听到理由吧）

忍者杀手 ニンジャスレイヤー杀／作者：Bradley Bond x Philip Ninj@ Morzez x 关根光太郎

由美国作家原作的科幻小说被翻译成日文，在日本推特上引起风潮的作品，虽然充斥了欧美人对〝忍者〞的误会（？）与幻想，但连日本人自己都不太介意了～应该也还好吧（？）

迷糊公务员／作者：高津カリノ

描述日本公务员的日常故事，或许对欧美读者来说真的有一些难以理解的笑点跟文化差异，阿珂是觉得还蛮有趣的说～

《野兽老师的女僕》最大卖点其实是作者的节省纸张术

发表于 2020-10-06

日本Yahoo! 最近的漫画推荐专栏，因为打错书名而意外成为网路话题，看来误植的效果可能比普通的介绍还有用？这本书名原本是「野獣先生のメイドさん」（野兽老师的女僕），结果日本Yahoo! 新闻出现的标题竟然是野兽前辈！野兽前辈出自2001年发行的男同志成人影片《仲夏夜之淫梦》，日本讨论版有不少成句、恶搞梗都和这部片子有关。看来野兽前辈的大名已经渗透到日本的各个角落了？(￣▽￣)

日本Yahoo! 专栏误植（目前已经更正）

这则文章原本是要讨论作者的省纸术，没想到焦点反而变成意料之外的野兽前辈，虽然成为瞩目焦点，但对作者来说不知道会不会觉得有点複杂？

活跃于niconico以及各大讨论版的野兽前辈

这部男同志成人影片《仲夏夜之淫梦》虽然距离发行时间已经有15年，不过影片中所使用的台词和桥段依旧常被拿来引用或恶搞。其中第四章登场的野兽前辈（剧中角色名为田所）就是人气角色之一。

作者「くりもとぴんこ」本月曾经在推特分享自己都是把分镜画在废纸上，某次想说纸张差不多快用完时，顺便随手抽了一张纸，赫然发现这张纸不但用过而且还是他高中时代的纸张！连老师自己都吓了一跳。

撇开误植问题，《野兽老师的女僕》本身的故事当然也很有趣，对反差萌有兴趣的读者或许会喜欢。剧情描述为了找打工而烦恼的女高中生「小森みみ子」突然被外表看似凶神恶煞的同班同学大神有人叫出去钉孤枝（误）其实大神是要介绍她女僕的工作。

动画宅想弃《orange橘色奇蹟》超大咖人设亲自留言解释

发表于 2020-10-06

已经圆满完结的少女漫画《orange橘色奇蹟》，先是在去年推出土屋太凤、山崎贤人主演的电影，接着今年7月开始又播出动画版，让粉丝可以透过动画再度体验全新的感动。《orange橘色奇蹟》虽然是以常见的「改变过去」为主题，不过最能引起粉丝共鸣的部分或许还是「想改变自己的学生时代」这点。再加上穿越时空的信件以及平行世界等设定带给读者无限的想像，让人忍不住想像如果自己也有机会改变过去的话该有多好。只是也有一些纯动画党的观众看了第一话后果断决定弃番，结果身为人设的超大咖动画师「结城信辉」居然亲自留言回覆！

以下有第一话剧透，还没看过的人请小心

女主角高宫菜穂在高二这年春天，收到一封自称十年后的她寄来的信，信上不但详细记载她这天发生的事情，甚至还提醒她这天千万不要在放学邀约转学生成瀬翔和他们一起回家。其实这一切都是为了要改变翔死亡的未来。

而第一话也透露十年后菜穗将会和弘人结婚生子，所以也有观众替弘人打抱不平。如果顺利拯救翔，那弘人未来与菜穗建立家庭的未来不就消失了！？这样对弘人来说也太残酷了。

推主「ぽろり春草 @syunsou」认为这部作品以女性的纯爱为名践踏男性的感情，所以就算动画製作水準很高，他也毅然决然决定弃番。

接着推主衍生出一个论点：因为不少女性都对目前身边的男伴抱着妥协的态度，即使满足于现在的生活，但如果有机会还是会想跟等级更高的男人交往。

没想到后来担任《orange橘色奇蹟》人设的结城信辉先生，亲自留言解释原作主旨其实不是如他所想的这样，一开始的确很容易让人误会没错，如果推主愿意的话希望他能继续看下去。

结城信辉从1983年起就在业界从事作画工作，曾担任过《圣天空战记》、《钢铁侦探J》 、《奔向地球》、《坂道上的阿波罗》人设兼作画监督。在业界的名声非常响亮。
